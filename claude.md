# kablau.co — Project Brief

## What is this?
A personal site. Gideon shares his own projects, thoughts, and experiences.

## Design Philosophy
- Minimal, intentional, lots of whitespace
- Dark background (#0f0f0e) — consistent across all pages
- Typography is the main graphic element
- Clean structure and hierarchy — the user always knows where to look
- Feels handcrafted, not like a generic tech blog

## Colors
- Background: #0f0f0e
- Surface: #1a1a18
- Text: #f0ede6
- Muted: #888780
- Border: rgba(255,255,255,0.08)

## Typography
- **Font: Neue Montreal** (from Pangram Pangram)
- Type scale based on a harmonic ratio
- Headlines are bold and confident
- Body text is comfortable to read

## Technical Stack
- Astro (static site)
- Hosted on Cloudflare Pages
- Connected to GitHub (gideonkablau/kablau-co)

## Rules for Claude
- Always keep design minimal — when in doubt, use more whitespace
- Never add unnecessary UI elements
- Typography first, decoration never
- Ask before adding new pages or sections
- Keep the codebase clean and readable
- Dark mode everywhere — never use light backgrounds
